# Dashboard Settings Specification

## Navigation
The top navigation bar contains the following items in order:
- "Dashboard" (links to /)
- "Settings" (links to /settings)
- "Help" (links to /help)

## Settings Page — Analytics Section
1. Navigate to the Settings page by clicking "Settings" in the top nav.
2. You will see a section titled "Analytics".
3. Inside that section there is a toggle labeled exactly "Enable Analytics".
4. Click the toggle to enable it.
5. A confirmation message appears at the bottom of the page reading exactly:
   "Analytics enabled successfully."

## Settings Page — Notifications Section
1. Inside the Settings page there is a section titled "Notifications".
2. There is a button labeled exactly "Save Preferences".
3. Clicking "Save Preferences" while no notification channels are selected
   shows an inline error: "Please select at least one notification channel."
