# Settings Page

The Settings page is accessible from the top navigation bar via the "Settings" link. It contains two sections: Analytics and Notifications.

## Navigation

The top navigation bar includes three links in this order: Dashboard, Settings, and Help. The Settings link navigates to the /settings route.

## Analytics Section

The Analytics section allows users to opt into product analytics tracking.

It contains a toggle labeled "Enable Analytics". When the toggle is turned on, a confirmation message appears at the bottom of the page reading "Analytics enabled successfully." When the toggle is turned off, the confirmation message disappears.

## Notifications Section

The Notifications section lets users configure how they receive alerts.

It contains a "Save Preferences" button. If the user clicks "Save Preferences" without selecting any notification channels, an inline error message appears: "Please select at least one notification channel."
